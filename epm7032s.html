Altera EPM7032S CPLD bitstream format

This is an automatically generated page. The corresponding generator can be found here. The raw bitstream might have to be extracted from an Altera Quartus POF file first. libalterpof may help with that.

The bitstream for the EPM7032S consists of 15033 bits in total.

Product terms

The device contains 2 Logic Array Blocks (LABs), each containing 16 macrocells.
Bits 0 through 14079 of the bitstream configure the device's product terms.
Each product term is configured by 88 bits. A '1' represents a disregarded signal, a '0' appends the signal to the product.
The lower 72 bits switch global signals ('G'), routed to the LAB from the PIA (36 signals, once inverted and once non-inverted).
The higher 16 bits switch regional foldback signals ('R'), i.e. outputs from macrocells of the same LAB.

Macrocell configuration

13 bits configure one macrocell. With 2 LABs this makes for 416 bits (52 bytes) in total for macrocell configuration.

PIA-to-LAB routing configuration

Every LAB can use 36 global signals by routing them to the LAB from the PIA.
A global signal is selected by enabling the corresponding switch/multiplexer: A '1' disables a PIA signal, a '0' enables it.
Caveat: Enabling multiple switches for one LAB signal may damage the device.
Every LAB signal can be selected from 2 choices. With one bit per choice this makes for 72 bits per LAB and 144 bits (18 bytes) in total for PIA to LAB routing configuration.
Assuming that every switch/multiplexer selects a different PIA signal, a maximum of 72 PIA signals are routable to a LAB.

I/O pin configuration

12 bits configure one I/O block. With 32 I/O blocks this makes for 384 bits (48 bytes) in total for I/O configuration.
